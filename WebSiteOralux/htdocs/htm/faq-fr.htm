    Généralités

    Le CD Oralux va t-il modifier mon disque dur ?

    Oralux peut fonctionner sans disque dur si votre PC possède suffisamment de mémoire. En principe, il ne modifie pas votre disque dur.

    Par contre, le CD peut automatiquement utiliser une partition de swap déjà présente sur le disque dur.

    Et si l'utilisateur a créé via le menu d'Oralux, un stockage persistant sur son disque dur, celui-ci sera automatiquement utilisé par Oralux au prochain démarrage du PC.

    Quellles sont les caractéristiques minimum du matériel ?

    Oralux fonctionne avec un Pentium 200 Mhz ayant 128 Mo de RAM.

    Moins de RAM est possible aussi, mais une partition de swap pourra être requise.

    Vérifier le fichier iso téléchargé [1]

    Comment vérifier que le fichier téléchargé est correct

    Note importante: veillez à télécharger le fichier iso en mode binaire (FTP). Par défaut, Netscape télécharge le fichier en mode ASCII ce qui corrompt l'image.

    Si le téléchargement échoue, utilisez "Resume" pour qu'il reprenne. Cela fonctionne avec wget, ftp (reget), ncftp, CuteFTP, Getright, ...

    Avec MS-Windows

    Vous contrôlerez l'intégrité du fichier iso grâce à la clé md5, téléchargez auparavant md5sum.exe et le fichier md5sum d'Oralux.

    Et avec GNU/Linux

    Le fichier md5sum d'Oralux sera placé dans le même répertoire que le fichier iso.

    Tapez alors, dans un shell:

    md5sum -c Oralux_0.7-alpha.md5

    Graver le CD à partir du fichier téléchargé [1]

    Comment graver le CD sous MS-Windows ?

    Tous les logiciels de gravage sont capables d'utiliser un fichier iso.

    Easy CD Creator
    • Lancez Easy CD Creator.
    • Allez à "File"
    • ensuite à "Create CD from disc image".
    • Modifier le bas du menu descendant ("Files of type:") par "ISO image files".
    • Il fera la suite une fois que vous aurez fourni le fichier *.iso.

    Merci à zacheyer.

    WinOnCD

    Vous pouvez sélectionner un projet "track-image" si vous cliquez sur le bouton New. Dès que vous êtes en mode piste (track), mais pas le mode file-system, alors si vous cliquer-déposer un fichier image dans la fenêtre CD, le logiciel devrait l'interpréter comme une piste de type raw. WinOnCD utilise .RAW pour extension.

    Merci Rick Jones.

    Nero Burning Rom
    • Lancez Nero.
    • Fermez le wizard s'il est démarré automatiquement.
    • Allez à "File"
    • ensuite à "Burn Image...".
    • Cliquez, dans le dialogue suivant, sur "Image Files (*.nrg)" et sélectionnez "All files (*.*)".
    • Choisissez le fichier iso puis acceptez.
    • Dans le dialogue suivant assurez-vous que :
      1. Type of Image: Data Mode 1
      2. Block size: 2048
      3. Image header: 0
      4. Image trailer: 0
      5. Toutes les cases sont décochées.
    • Cliquez sur "OK".
    • Vérifiez dans le dialogue suivant que les options "write" et "finalize" sont sélectionnées.
    • Cliquez sur "Write".

    Merci Antonio Asensi.

    Et sous GNU/Linux ?

    Tout d'abord, déterminez les paramètres scsi de votre graveur de CD.

    cdrecord dev=ATAPI -scanbus

    Si les paramètres scsi sont 0,0,0 et la vitesse d'enregistrement 8x, vous taperez

    cdrecord -v -eject speed=8 -pad dev=ATAPI:0,0,0 oralux.iso

    Démarrage d'Oralux

    Comment démarrer le CD ?

    Votre PC au démarrage devra lancer le CD. Le BIOS du PC sera configuré pour démarrer de préférence le CDROM avant par exemple le système d'exploitation se trouvant sur le disque dur.

    Ou bien, une autre possibilité est de démarrer sur une disquette qui lancera le CD Oralux.

    Pourquoi y a t-il une attente de 30 secondes avant le démarrage d'Oralux ?

    Quand le PC démarre sur le CD, pendant quelques secondes la ligne de saisie suivante sera affichée :

    boot:

    Cette ligne permet de donner des indications dans le cas où l'auto-détection du matériel ne se passerait pas bien.

    Les mots clés attendus sont appelés "cheat codes".

    Si rien n'a été saisi au bout de 30 secondes, Oralux démarre automatiquement.

    Qu'entend t-on au tout début ?

    Vous devriez entendre le chant du coq Cocoralux suivi d'un beep.

    Si vous n'entendez rien, il y a plusieurs possibilités:

    • Il y a conflit entre la carte son et le composant audio présent sur la carte mère. Une solution est de désactiver le composant audio via le bios.
    • La détection de votre carte son a échoué: essayez de redémarrerez le PC en utilisant le bon mot clé.
    • Le volume doit être simplement augmenté

    Quels sont les bureaux possibles ?

    La version 0.7 propose trois bureaux : Emacspeak, un environnement basé sur YASR, et Speakup.

    Vous sélectionner le bureau grâce au menu audio.

    Son

    Comment monter le son ?

    Juste après le chant du coq, Oralux émet un beep. Vous pouvez alors modifier le volume sonore :

    • Pour monter le son, appuyez sur la touche entrée.
    • Pour baisser le son, appuyez sur la touche espace.
    • Et sur toute autre touche pour passer à l'étape suivante.

    Le son ne fonctionne pas !

    Vous redémarrerez le PC en utilisant le "cheat code" oss.

    Lorsqu'aucun code n'est indiqué, les pilotes ALSA sont utilisés.

    En indiquant le code oss, ce sont les pilotes OSS qui seront choisis et ils sauront (peut-être) gérer le son de votre ordinateur.

    Dans la ligne de commande du boot, saisissez :

    knoppix oss

    Comment modifier le son sous Emacspeak ?

    Il y a différentes façons de modifier le volume. En voici une :

    ESCAPE ! aumix -v 50

    Cela signifie appuyer sur la touche ESCAPE, puis sur la touche !, et ensuite taper aumix -v 50.

    50 est une valeur moyenne. Les valeurs sont comprises entre 0 (le silence) et 100.

    ESCAPE ! permet de lancer une commande en ligne. Cette commande pourrait être comparé à MSDOS, mais c'est du shell (plus puissant).

    Ici la commande est simplement :

    aumix -v 50

    Cette commande lance le mixer aumix en lui demandant de mettre le volume à 50.

    Braille

    Oralux peut-il piloter un terminal braille ?

    Oui, grâce à BRLTTY.

    Votre afficheur braille peut être sélectionné grâce au menu audio.

    Ou bien au boot, utilisez le "cheat code" brltty.

    Par exemple :

    brltty=bm,ttyS0,de

    Les cheat codes

    Qu'est-ce qu'un "cheat code"

    Il s'agit d'un ou de plusieurs mots clé à fournir au démarrage qui permet de choisir un pilote ou de le configurer.

    C'est à l'usage, en fonction de votre matériel, que vous vous apercevrez si un mot clé doit être fourni ou non.

    Ces mots clés sont saisis au clavier ou bien transmis par la disquette de démarrage.

    Dans la ligne de commande du boot, précéder du prompt "boot:", vous entrerez par exemple :

    knoppix speakup_synth=ltlk

    Comme par défaut, le clavier n'est pas configuré pour le français, je dois taper la touche q au lieu du a:

    knoppix speqkup_synth=ltlk

    Liste des "cheat code"

    La liste se trouve sur le CD en /KNOPPIX/oralux-cheatcodes.txt.

    La disquette de démarrage

    Dans quels cas une disquette de démarrage est-elle recommandée ?

    En principe, l'ordinateur démarre directement Oralux.

    Mais parfois, le PC ne sait pas démarrer le CDROM ou bien l'utilisateur ne peut pas configurer facilement le BIOS.

    La disquette est très utile aussi pour fournir à Oralux, les "cheat code" Knoppix.

    A chaque démarrage, elle nous évite de saisir les codes.

    Comment générer cette disquette ?

    Sous MS-Windows

    sbm.bat est un fichier batch MSDOS qui permet la création d'une disquette de démarrage (Smart Boot Manager).

    Ce fichier se trouvent sur le CD dans le répertoire tools/cd-boot.

    Sous Oralux

    Insérez une disquette et tapez :

    dd if=/cdrom/tools/cd-boot/sbm.img of=/dev/fd0 bs=18k

    Qu'est ce que le logiciel DECtalk ?

    Si vous avez acheté le logiciel DECtalk, Oralux devrait pouvoir l'installer à partir de votre disque dur.

    DECtalk est disponible en plusieurs langues et en particulier en français.

    Nous n'avons bien sûr aucun intérêt financier à parler du logiciel DECtalk.

    Comment installer le logiciel DECtalk ?

    Afin que le CD puisse trouver votre fichier DECtalk, vous devrez créer un répertoire dans l'une de vos partitions. Mais pas une partition NTFS !

    Le nom du répertoire sera dtk

    Tous les caractères sont en minuscules.

    Ce répertoire devra se trouver au plus au niveau de l'arborescence de la partition.

    Par exemple, si /dev/hda1 est monté sur /var, Oralux réussira à trouver /var/dtk, mais échouera avec /var/foo/dtk .

    Placer, svp, dans votre répertoire dtk, le fichier DECtalk : le nom attendu est quelque chose comme linux...tar.gz. Si vous avez un second fichier DECtalk, placer le également dans le répertoire. Oralux peut installer le fichier optionnel en français ou bien celui en allemand ou en espagnol castillan.

    Oralux peut-il piloter une synthèse vocale externe ?

    Oralux inclut les pilotes de plusieurs synthèses matérielles pour Emacspeak ou Speakup.

    Si la carte son de votre PC est bien détectée, vous pourrez sélectionner vos préférences par le menu audio.

    Vous choisirez ainsi votre synthèse matérielle dans la liste proposée.

    Ma carte son n'est pas reconnue, comment lancer directement ma synthèse externe avec Emacspeak ?

    Le cheatcode tts

    Par exemple, la ligne suivante, saisie au boot, lance Emacspeak avec la synthèse DoubleTalk LT:

    knoppix tts=doubletalk

    Par défaut, la synthèse est supposée se trouver sur le premier port de communication (ttyS0).

    Pour indiquer plutôt le second port série, vous taperez :

    knoppix tts=doubletalk,ttyS1
    Pilotes disponibles

    Voici la liste des pilotes présents pour Emacspeak.

    Il y a tout d'abord le mot clé pour le cheatcode tts, puis l'appellation de la synthèse externe.

    • accent: Accent SA
    • braillenspeak: Braille Lite, Braille 'n Speak, Type 'n Speak
    • ciber: Ciber 232, Ciber 232 Plus
    • dtk-exp: DECtalk Express
    • dtk-mv: DECtalk Multivoice, DECtalk 3
    • apollo: Dolphin Apollo
    • doubletalk: DoubleTalk LT, LiteTalk
    • doubletalkpc: DoubleTalk PC
    • pchablado: PC Hablado notebook

    Comment lancer ma synthèse externe avec Speakup ?

    Le cheatcode speakup_synth

    Tapez s'il vous plaît, à l'invite du boot, pour sélectionner le pilote doubletalk lite :

    knoppix speakup_synth=ltlk
    Pilotes disponibles

    Cheatcode de synthèses externes (issu de spkguide.txt)

    • acntsa -- Accent SA
    • acntpc -- Accent PC
    • apolo -- Apolo
    • audptr -- Audapter
    • bns -- Braille 'n Speak
    • dectlk -- DecTalk Express (ancien et récent, seulement le db9 série)
    • decext -- (Ancien) DecTalk Externe
    • dtlk -- DoubleTalk PC
    • keypc -- Keynote Gold PC
    • ltlk -- DoubleTalk LT, LiteTalk, ou Tripletalk externe (seulement db9 série)
    • spkout -- Speak Out
    • txprt -- Transport

    Note: Speakup ne supporte * PAS * les connexions USB ! Speakup ne supporte * PAS * non plus le Tripletalk interne !

    Comment choisir la vitesse de la voix sous Emacspeak ?

    Pour modifier la vitesse de la voix:

    C-e d 1

    1 pour une voix très lente,

    C-e d 9

    9 pour une voix très rapide.

    C-e signifie : maintenez appuyée la touche CONTROL, et appuyez sur la touche e. Relâchez les deux touches. Appuyez sur la touche d, puis sur 9.

    Toute commande Emacspeak commence avec C-e. d correspond au serveur vocal (le premier a été DECtalk).

    Sauvegarder ses préférences et ses documents

    Comment sauvegarder mes Préférences ?

    Vous souhaiterez sûrement sauvegarder le volume, le type de clavier, en bref toutes les préférences sélectionnées dans le menu d'introduction.

    Idem pour les autres fichiers de configuration comme par exemple les paramètres internet.

    Saveconfig permet de sauvegarder ces fichiers sur un un disque dur, une disquette, une clé USB,...

    Lors des redémarrages suivants, ces paramètres seront automatiquement recherchés.

    saveconfig peut être automatiquement lancé à partir du menu audio (le choix "stockage persistant").

    Comment stocker mes Documents ?

    Le répertoire où vous pourrez facilement écrire des fichiers se trouve en /home/knoppix.

    Par défaut, ce répertoire se trouve dans la mémoire non sauvegardée de votre ordinateur. Dés que l'ordinateur sera éteint ces données seront perdues !

    En fait, si vous le souhaitez, le répertoire /home/knoppix sera sauvegardé dans une 'image' sur votre disque dur ou votre clé USB.

    Au redémarrage suivant, cette image sera automatiquement recherché.

    Cette image peut être automatiquement créée à partir du menu audio (le choix "stockage persistant").

    Arrêter le système

    Comment arrêter le PC ?

    Quittez tout d'abord Emacspeak grâce à C-x C-c ou bien quitter le menu sous YASR ou Speakup grâce a la touche Échappement.

    Ensuite, le menu audio vous proposera l'arrêt du PC.

    Et pour le redémarrer ?

    Quittez tout d'abord Emacspeak grâce à C-x C-c ou bien quitter le menu sous YASR ou Speakup.

    Ensuite, grâce au menu audio, redémarrer le PC.

    Ou bien appuyez sur les trois touches magiques Ctrl Alt Del

    Questions diverses.

    Comment installer Oralux sur disque dur ?

    L'installation sur disque dur est une caractéristique à venir.

    Néanmoins, knoppix-installer est disponible, à vos propres risques, pour installer le CD sur le disque dur. Consultez la liste de diffusion pour obtenir les renseignements les plus récents.

    Une autre possibilité est de copier le fichier Oralux compressé du CD vers une partition vfat ou ext2 (mais pas ntfs). Moins de 700 Mo seront utilisés. Oralux sur le disque dur se comportera alors comme sur le CD: il ne sera pas possible d'ajouter ou de modifier les programmes. Cela peut être un avantage aussi.

    Les deux étapes sont :

    1. Créer un répertoire KNOPPIX. Ce répertoire sera au plus niveau. Par exemple, avec MS-Windows, C:\KNOPPIX est correct alors que C:\MonDossier\KNOPPIX ne fonctionnera pas.
    2. Copier le fichier KNOPPIX/KNOPPIX du CD dans le nouveau répertoire.

    Ensuite, il faudra démarrer le PC avec le CD Oralux ou une disquette de boot présents pour qu'ils lancent Oralux stocké sur disque dur.

    Pourquoi un coq comme logo ?

    Effectivement, le logo d'Oralux est un coq chantant devant le soleil.

    Le chant du coq rappelle le mot Oral, et comme le coq chante au lever du soleil, cela rappelle Lux qui signifie Lumière.

    Et puis, ce n'est pas n'importe quel coq, c'est Cocoralux !

    Documentation

    Conventions

    Juste quelques mots, si vous n'êtes pas familiarisé avec les syntaxes Emacs C-x ou M-x.

    C-x signifie: maintenez appuyée la touche CTRL puis appuyez sur la touche x (la touche CTRL étant toujours appuyée).

    M est appelé touche META : c'est la touche gauche ALT.

    M correspond également à l'appui sur la touche Échappement. La touche Échappement n'est pas maintenue appuyée lors de l'appui de la touche suivante.

    Info

    Les documentation Emacs ou Emacspeak sont principalement au format Info. Ce format structure la documentation, permet de se déplacer de noeud en noeud. Pour apprendre rapidement les quelques commandes Info, tapez juste C-h i
