Traduction d'Oralux

Si vous le souhaitez, vous pouvez nous aider à traduire les nouveaux messages ou documentations d'Oralux.

Actuellement, les langages possibles sont l'allemand, l'anglais, l'espagnol, le français, et le russe.

Les liens ci-dessous vous feront accéder aux pages de traduction.

Ces pages sont gérées par un wiki, ainsi vous pourrez plus facilement transmettre vos traductions en ligne. Voici les différentes étapes :

  • Sélectionner ci-dessous une page de traduction,
  • Dans la page, sélectionner le lien "Modifier",
  • La page s'affiche alors avec une zone de saisie,
  • Tapez votre traduction,
  • Puis transmettez-la grace au bouton "Transmettre". Attention, si vous oubliez d'appuyer sur le bouton "Transmettre", tout votre travail sera perdu !
