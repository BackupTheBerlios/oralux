Respostas às perguntas mais freqüentes:

Geral

O Oralux mexe no meu disco rígido ?

Se a máquina possuir memória RAM suficiente, o Oralux funciona perfeitamente sem disco rígido. Neste caso, nada que esteja no disco é afetado.

Não obstante, o CD pode usar alguma partição swap que já exista no disco rígido.

Caso também o usuário crie, usando o audiomenu do Oralux, um local de armazenamento permanente no disco, esse local será usado pelo Oralux nas próximas vezes que iniciar.

Qual o hardware mínimo suportado ?

Recomenda-se um Pentium 200 Mhz com 128 MB de RAM.

Se essa RAM for menor, provavelmente será necessária uma partição swap.

Verificando o arquivo iso baixado[1]

Como verifico se o arquivo iso que baixei está completo ?

Importante: Certifique-se de baixar o ISO em modo binário (FTP). Por padrão, o Netscape baixa em modo ASCII e isso corrompe a imagem.

Se o download for interrompido, você pode lançar mão do "Resume" que permite continuá-lo de onde parou. Isso funciona no wget, ftp (reget), ncftp, CuteFTP, Getright, etc.

No MS-Windows:

Uma vez baixado o arquivo, você pode verificar a integridade do mesmo usando o código md5. para tanto baixe o md5sum.exe e o código md5sum do CD.

No GNU/Linux

O código md5sum deve ser posto no mesmo diretório do arquivo iso.

No prompt digite:

md5sum -c Oralux_0.7.md5

Gravando o CD do Oralux a partir do arquivo iso[1]

no MS-Windows

Qualquer programa para gravação de CD pode gravar imagens iso em CD-Rom.

Easy CD Creator
  • Abra o Easy CD Creator.
  • Aceda ao menu "File".
  • Escolha "Create CD from disc image".
  • Na caixa de diálogo, certifique-se de selecionar, na caixa de combinação "Arquivos do tipo:" a opção "Arquivos de imagem ISO".
  • Aí clique no nome do arquivo iso e o programa encarrega-se do resto.

Agradecimentos: Zacheyer.

WinOnCD

Clique no botão "New" e escolha um projeto "track-image". Quando estiver no modo track e não no modo file-system, ao arrastar um arquivo de imagem até a janela do CD, o programa trata-o como uma faixa raw. O WinOnCD usa .RAW como extensão própria.

Agradecimentos: Rick Jones.

Nero Burning Rom
  • Abra o Nero.
  • Feche o assistente caso abra automaticamente.
  • Clique em "Gravar Imagem...".
  • E então em "Gravar Imagem...".
  • Na caixa de diálogo, clique em "Arquivos de imagem (*.nrg)" e escolha "Todos os arquivos (*.*)".
  • Encontre o arquivo .iso, clique nele e dê OK.
  • Na próxima caixa de diálogo, certifique-se que os seguintes parâmetros estejam ajustados:
    1. Tipo de imagem: imagem de dados 1
    2. Tamanho do bloco: 2048
    3. Cabeçalho da imagem: 0
    4. Image trailer: 0
    5. Todas as caixas de seleção desmarcadas.
  • Clique em "OK".
  • Na próxima janela, certifique-se que as opções "gravar" e "finalizar CD" estejam marcadas.
  • Clique em "Gravar".

Agradecimentos: Antonio Asensi.

no GNU/Linux

Primeiro configure o gravador SCSI de destino. Digite:

cdrecord dev=ATAPI -scanbus

Se o destino SCSI for 0,0,0 e a velocidade for 8x, digite:

cdrecord -v -eject speed=8 -pad dev=ATAPI:0,0,0 oralux.iso

Iniciando o Oralux

Como iniciar o Oralux?

O computador iniciará automaticamente a partir do CD, caso a Bios esteja configurada para isso. É necessário que ela esteja programada para buscar na unidade de CD ROM algum disco de inicialização, antes de iniciar por uma mídia não-removível como por exemplo o sistema operacional instalado no disco rígido.

Outra possibilidade é iniciar por um disquete que então inicialize o CD. Existem arquivos de lote (.bat) para criação de disquetes inicializáveis no CD do Oralux, diretório /Knoppix.

Por que os 30 segundos de espera antes de o Oralux iniciar ?

Quando o computador inicia pelo CD, o seguinte prompt de comando é mostrado na tela por alguns segundos:

boot:

Por meio desse prompt você pode informar uma série de parâmetros de inicialização, como por exemplo o tipo do seu display braille caso disponha de um. Esses parâmetros são conhecidos como "cheat codes".

Caso nada seja digitado nesses 30 segundos, o Oralux inicia automaticamente. Se não quiser aguardar até o Oralux iniciar, pressione "enter" no prompt de "boot:".

Que som ouvirei primeiro ?

Se tudo correr bem, você ouvirá o mascote do Oralux, o "Galoralux", cantar. Em seguida soa um bipe.

Caso não haja som algum, as causas podem ser várias:

  • O computador pode ter uma placa de som e também um chip de áudio onboard. Uma solução nesse caso é desabilitar na Bios o chip onboard.
  • A detecção do dispositivo de som pode ter falhado; nesse caso você deve reiniciar e informar o "cheat code" apropriado.
  • O volume precisa ser aumentado.

Quais os ambientes de trabalho disponíveis ?

A versão 0.7 oferece três ambientes: O Emacspeak, um ambiente baseado no YASR, e o Speakup.

Pode escolher o ambiente que deseja usando o audiomenu.

Som

Como aumento o volume ?

Quando o galo pára de cantar o Oralux bipa. Aí você pode alterar o volume:

  • Para aumentar, pressione Enter.
  • Para diminuir, pressione barra de espaço.
  • Qualquer outra tecla pula para a próxima etapa.

Mas não tem som nenhum ! E agora ?

Reinicie e use o cheat code oss.

Quando nenhum parâmetro é informado, os drivers ALSA são carregados.

Se for informado o parâmetro oss, os drivers OSS serão carregados e espera-se que estes consigam controlar a contento o dispositivo de som.

Digite o seguinte comando no prompt de início:

knoppix oss

Como altero o volume no Emacspeak ?

Existem várias maneiras; Eis uma:

ESCAPE ! aumix -v 50

Isto é: tecle ESCAPE, o símbolo ! e digite "aumix -v 50".

50 é o volume médio. Você pode colocar qualquer valor entre 0 (silêncio) e 100 (volume máximo).

As teclas ESCAPE e ! em seqüência abrem um prompt no qual deve-se digitar um comando. Esse prompt é semelhante àquele do MS-DOS, porém trata-se de um shell, bem mais poderoso portanto.

Aí o comando é:

aumix -v 50

Esse comando chama o mixer "aumix" e coloca o volume em 50.

Braille

O Oralux trabalha com displays braille ?

Sim, graças à "BRLTTY".

você pode escolher seu display braille no audiomenu.

Pode ainda durante o boot, usar o "cheat code" brltty.

Exemplo:

brltty=bm,ttyS0,de

Parâmetros Cheat codes

O que é um "cheat code"?

É um parâmetro passado durante o boot, que serve como auxílio na detecção e/ou configuração de drivers necessários ao hardware específico do computador.

À medida que for adquirindo experiência, você encontrará os cheat codes mais adequados à sua máquina quando se fizerem necessários.

Tais parâmetros podem ser digitados ou buscados automaticamente num disco de inicialização que contenha as devidas configurações pessoais.

Durante a inicialização, é mostrado por 30 segundos este prompt: "boot:". Você pode teclar, por exemplo:

knoppix speakup_synth=ltlk

Lista dos cheat codes

Essa lista encontra-se no CD, em /KNOPPIX/oralux-cheatcodes.txt.

Disco de Inicialização

Quando é uma boa iniciar por um disquete ?

Geralmente o computador consegue iniciar pelo CD do Oralux de modo automático.

Alguns poucos PCs porém não conseguem fazê-lo e, em outros tantos casos, o usuário depara-se com muitas dificuldades para configurar corretamente a BIOS.

Um disquete pode também conter parâmetros que sejam necessários toda vez que o computador inicia, como por exemplo os cheat codes, que doutro modo teriam de ser digitados em cada uma dessas ocasiões.

Como crio um disco de inicialização ?

No MS-Windows

O sbm.bat é um arquivo de lotes do MS-DOS capaz de criar o disco, usando o Smart Boot Manager.

diretório: tools/cd-boot

No Oralux

Insira um disquete e digite:

dd if=/cdrom/tools/cd-boot/sbm.img of=/dev/fd0 bs=18k

O que é o software DECtalk ?

Se você comprou o software DECtalk, o Oralux pode instalá-lo a partir do disco rígido.

O DECtalk existe em seis idiomas, especialmente em Francês.

Logicamente não temos qualquer interesse financeiro em divulgar o referido software.

Como instalar o DECtalk ?

Para que o Oralux encontre o DECtalk, você tem que criar um diretório numa das partições do disco rígido. Não o faça em partições NTFS porque o Oralux ainda não consegue escrever nessas partições e se o pacote for lá colocado, o número de série do DECtalk será pedido a cada início do Oralux.

O diretório tem que se chamar dtk (tudo em minúsculo).

Ele deve cituar-se no topo da árvore daquela partição.

Como exemplo, se /dev/hda1 estiver montado em /var o Oralux encontrará /var/dtk mas não encontrará /var/foo/dtk .

Coloque o pacote DECtalk no diretório ddk; o nome dele é algo como linux...tar.gz

Se você tem outros pacotes DECtalk, coloque-os ali também e o Oralux pode então instalar como opção os idiomas Francês, Espanhol Castelhano e Alemão.

O Oralux trabalha com sintetizadores externos ?

Estão inclusos drivers para os sintetizadores que funcionam com o Emacspeak e o Speakup.

Se sua placa de som funciona no Oralux, você pode configurar as preferências que quiser pelo audiomenu.

Como exemplo, você pode escolher o sintetizador externo que deseja a partir de uma lista.

Minha placa de som não funciona. Como usar um sintetizador externo no Emacspeak ?

O parâmetro tts

Se entrar por exemplo com a seguinte linha durante a inicialização, o Emacspeak é iniciado com o DoubleTalk LT:

knoppix tts=doubletalk

Por padrão supõe-se que o sintetizador esteja conectado à primeira porta serial (ttyS0).

Para escolher a segunda porta entre com:

knoppix tts=doubletalk,ttyS1
Drivers disponíveis

Eis a lista de drivers para Emacspeak.

À esquerda está o parâmetro tts e à direita o sintetizador a que este se refere.

  • accent: Accent SA
  • braillenspeak: Braille Lite, Braille 'n Speak, tipo 'n Speak
  • ciber: Ciber 232, Ciber 232 Plus
  • dtk-exp: DECtalk Express
  • dtk-mv: DECtalk Multivoice, DECtalk 3
  • apollo: Dolphin Apollo
  • doubletalk: DoubleTalk LT, LiteTalk
  • doubletalkpc: DoubleTalk PC
  • pchablado: PC Hablado notebook

Como iniciar meu sintetizador externo no Speakup ?

O parâmetro speakup_synth

Para escolher o driver do doubletalk lite digite:

knoppix speakup_synth=ltlk
Drivers disponíveis

Parâmetros para sintetizadores externos (fonte: spkguide.txt)

  • acntsa -- Accent SA
  • acntpc -- Accent PC
  • apolo -- Apolo
  • audptr -- Audapter
  • bns -- Braille 'n Speak
  • dectlk -- DecTalk Express (antigo e atual, somente db9 serial)
  • decext -- DecTalk (antigo) External
  • dtlk -- DoubleTalk PC

Como altero a velocidade da voz no Emacspeak ?

Tecle:

C-e d 1

1 = Velocidade muito lenta,

C-e d 9

9 = Velocidade muito rápida.

Experimente algarismos entre 1 e 9 até achar a velocidade que mais lhe agrade.

Nota: C-e significa apertar e manter pressionada a tecla CONTROL, teclar e, soltar as duas teclas, teclar d e depois 9.

Todos os comandos do Emacspeak começam com C-e. A letra d refere-se ao servidor de fala (neste caso o primeiro é o DECtalk).

Salvando preferências e documentos em local permanente

Como salvo minhas preferências ?

Provavelmente você deseje salvar o volume, o tipo de teclado e outras configurações que tenha feito por meio do menu principal e também outros arquivos como por exemplo os parâmetros de conexão à Internet.

O Saveconfig pode salvar tudo no disco rígido, num disquete ou nalgum drive USB.

Assim essas configurações serão automaticamente procuradas e carregadas nos próximos inícios do Oralux.

O saveconfig pode ser automaticamente chamado a partir do audiomenu, item "persistent storage".

Como salvo meus documentos ?

O diretório onde você pode guardar facilmente os documentos é o /home/knoppix.

Por padrão esse diretório fica na memória RAM, o que quer dizer que cada vez quando desligar o computador, perderá os dados ali presentes !

Caso deseje, pode salvar o diretório /home/knoppix em uma "imagem" no disco rígido ou num drive USB.

Quando iniciar novamente o Oralux, essa imagem será procurada de forma automática.

A imagem pode ser criada no audiomenu, item "persistent storage"

Finalizando

Como desligar meu PC ?

Feche o Emacspeak teclando C-x C-c

Ou, feche o YASR ou o menu do Speakup teclando Escape.

O audiomenu proporá então o desligamento do PC

E como reiniciar ?

Feche o Emacspeak teclando C-x C-c, ou, feche o YASR ou o menu do Speakup.

Aí, no audiomenu, reinicie o PC.

Alternativamente, pressione as três teclas mágicasControl Alt Del

Diversas

Como instalar o Oralux no disco rígido ?

A instalação real em disco é um recurso ainda a ser implementado.

Não obstante, existe o "knoppix-installer" que você pode usar por conta e risco próprios a fim de instalar o Oralux no disco. Por favor consulte a lista de discussão do projeto para obter informações atualizadas.

Outra opção é copiar o arquivo comprimido do CD para o disco rígido, em qualquer partição vfat or ext2, nunca em NTFS. São ocupados menos de 700 MBytes. Nesse caso o Oralux comportar-se-á exatamente como no CD: não será possível instalar e atualizar programas, o que talvez seja uma vantagem.

Eis as duas etapas dessa operação:

  1. Crie o diretório knoppix no topo da árvore da partição. Por exemplo, no MS-Windows, C:\KNOPPIX está certo; C:\MinhaPasta\KNOPPIX está errado.
  2. Copie o arquivo KNOPPIX/KNOPPIX do CD para o diretório recém-criado.

Será necessário o CD do Oralux ou um disco de inicialização para iniciar o Oralux do disco rígido.

Por que um galo como mascote ?

Realmente, o logotipo do Oralux é um galo cantando de frente para o sol.

O canto do galo vai escrevendo a palavra Oral e conforme o galo vai cantando ao pôr do sol, vai sendo escrito Lux que significa luz.

E ele não é um galo qualquer, é o Galoralux !

Convenções

Algumas notas caso a sintaxe C-x ou M-x do Emacs não lhe seja ainda familiar.

C-x significa apertar e segurar a tecla Control, teclar x e soltar ambas.

M é a assim chamada "META key" e corresponde ao alt esquerdo na maioria dos teclados.

A tecla Escape funciona igualmente como M, porém não deve ser mantida presa ao apertar a outra tecla.
