Comprobando el archivo iso descargado [1]

¿Cómo puedo comprobar que el archivo iso que descargué es correcto?

Nota importante: ten cuidado de descargar el archivo iso en modo binario (FTP). Por defecto, Netscape descarga el archivo en modo ASCII lo que puede corromper la imagen.

Si la descarga falla puedes usar la opción "Reanudar" que te permite continuarla. Esta opción funciona con wget, ftp (reget), ncftp, CuteFTP, Getright, ...

Usando MS-Windows

Puedes comprobar la integridad del archivo ISO con la clave md5, descargando previamente md5sum.exe y el archivo Oralux md5sum.

Y con GNU/Linux

El archivo Oralux md5sum puede colocarse en el mismo directorio en el que está el archivo iso.

Entonces escribe, en una terminal shell:

md5sum -c Oralux_0.6-alpha.md5

Grabando el CD desde el archivo descargado [1]

¿Cómo grabar el CD bajo MS-Windows?

Todos los programas para escritura de CD pueden usar imágenes ISO para grabarla directamente al CD-Rom.

Easy CD Creator
  • Abre Easy CD Creator.
  • Ve a "Archivo".
  • luego a "Crear CD a partir de imagen de CD".
  • Asegúrate de cambiar el menú inferior de la ventana emergente ("Tipo de archivos:") para que diga "Archivos imagen ISO (*.iso)".
  • Esto debiera hacer el resto después que busques el archivo *.iso.

Gracias a Zacheyer.

WinOnCD

Puedes elegir una "pista de imagen" cuando elijas el botón nuevo. Estando en modo pista, no en modo sistema de archivos, arrastra el archivo de imagen a la ventana del CD que debería aceptarla como una pista "raw". WinOnCD usa .RAW como su propia extension.

Gracias a Rick Jones.

Nero Burning Rom
  • Abre Nero.
  • Cierra el asistente que se abre automáticamente.
  • Ve a "archivo".
  • Luego a quemar imagen...
  • En el próximo diálogo, ve a archivos de imagen (*.rng) y elige "todos los archivos" (*.*)".
  • Elige el .iso y acepta.
  • En el siguiente diálogo asegúrate:
    1. El tipo de imagen es "modo de datos 1"
    2. Tamaño de bloque: 2048.
    3. Image header: 0
    4. Image trailer: 0
    5. Todos los botones de opción están verificados.
  • Elige aceptar.
  • En el próximo diálogo comprueba que "escribir" y "finalizar" están activadas.
  • Elige "escribir".

Gracias a Antonio Asensi.

¿Y bajo GNU/Linux?

Primero, determina el indicador SCSI del Grabador de CD:

cdrecord -scanbus

Si el indicador SCSI es 0,0,0 y la velocidad 8x, puede que quieras lanzar:

cdrecord -v -eject speed=8 -pad dev=0,0,0 oralux.iso

Iniciando Oralux

¿Cómo iniciar Oralux?

Tu PC puede iniciar desde el CD. La BIOS ha de estar configurada para que el CD arranque antes que, por ejemplo, el sistema operativo en el disco duro.

Otra opción es arrancar desde un disquete que inicie el CD de Oralux.

¿Por qué esos 30 segundos de espera antes de que se inicie Oralux?

Cuando el PC arranca desde el CD, aparece la siguiente entrada de línea de comandos durante unos segundos:

boot:

Usando esta línea de comandos, puedes introducir el tipo de terminal braille que tienes. Las palabras clave que puedes introducir se llaman "cheat codes".

Si pasados 30 segundos no se presiona ninguna tecla, Oralux se inicia automáticamente.

¿Qué oiré al comienzo?

Deberías oir cantar al gallo Galloralux seguido de un pitido.

Si no hay sonido, hay dos posibles causas:

  • O la detección de tu dispositivo de sonido ha fallado; debes arrancar otra vez proporcionando el "cheat code" apropiado.
  • O simplemente debes aumentar el volumen.

¿Cuales son los posibles escritorios?

La versión 0.6 propone dos escritorios: Emacspeak o un ambiente basado en YASR.

Quizás quieras seleccionar el escritorio usando el menú de audio.

Cuando Emacspeak arranca, un corto mensaje es desplegado y se suministra mas detalles sobre el escritorio de audio.

YASR es un lector de pantalla. Bajo YASR, tu puedes seleccionar un programa y correrlo usando un "mini menú".

Estos programas son:

  • ne, un intuitivo editor de texto. ne significa 'Nice Editor'.
  • links 2, el navegador web con soporte para javascript y un modo de funcionamiento para personas con discapacidad visual (numeración de links,...).
  • una calculadora (bueno, de hecho es python.)
  • bash, la interfaz para la línea de comandos.

Este mini menú será actualizado y mejorado en el futuro.

Sonido

¿Cómo subir el volumen al sonido?

Una vez el gallo termine de cantar, Oralux emite un pitido. Entonces, puedes cambiar el volmen :

  • Para aumentar el volumen, presiona la tecla Enter.
  • Para disminuir el volumen, presiona la tecla espacio.
  • Y cualquier otra tecla para pasar a la siguiente etapa.

¡No hay sonido!

Reinicia el PC y usa el "cheat code" de alsa.

Cuando no se proporciona ningún "cheat code", se usan los drivers OSS.

Proporcionando la palabra clave alsa, se usan los drivers ALSA, esperando que sean capaces te manejar tu dispositivo de sonido.

Introduce la siguiente línea de commandos en el momento del arranque:

knoppix alsa

¿Cómo cambiar el sonido bajo Emacspeak?

Para cambiar el volumen, hay varios métodos. Aquí hay uno:

ESCAPE ! aumix -v 50

Significa presionar la tecla ESCAPE, después la tecla ! y por último teclear aumix -v 50.

50 es el valor medio. Puedes escoger cualquier valor entre 0 (silencio) y 100.

ESCAPE ! abre un linea de entradas, donde espera una linea de comandos. Esta línea de comandos podría compararse con MSDOS, pero ésta es una shell (más potente). Aquí, la linea de comando es simplemente:

aumix -v 50

Este comando llama al mezclador aumix y regula el volumen a 50.

Braille

¿Es Oralux capaz de manejar una línea braille?

Si, gracias a BRLTTY

Desde la versión 0.6, tu interfaz braille puede ser seleccionada usando el menú de audio.

O en el momento de arranque, usando el "cheat code" brltty.

Por ejemplo:

brltty=al,/dev/ttyS0,text.german.tbl

Cheat codes

¿Qué es un "cheat code"?

Es una palabra clave suministrada en el momento del arranque que ayuda a seleccionar o configurar los controladores.

A través de la práctica descubrirás si los "cheat codes" son necesarios para tu ordenador.

Estas palabras clave son tecleadas o suministradas mediante el uso de un disquete de arranque.

En el momento del arranque, durante 30 segundos, se muestra este aviso: "boot:". Puede que quieras teclear por ejemplo

knoppix alsa

Cómo guardar el "cheatcode" de alsa

Oralux puede guardas el código de alsa. Así, para los próximos arranques, no será necesario teclearlo de nuevo.

Aquí está la secuencia:
Teclea en el momento del arranque el "cheat code" de alsa.
Una vez que Emacspeak se ha iniciado, corre saveconfig para guardar tu configuración.
Para los próximos arranques, el "cheatcode" de alsa será restablecido automáticamente.

Para eliminar el código de alsa, teclea el "cheatcode" noalsa en el momento del arranque:

knoppix noalsa

Después, una vez el Emacspeak se haya iniciado, corre saveconfig.

Lista de "cheat code"

La lista está dentro del CD en /KNOPPIX/oralux-cheatcodes.txt.

Disquete de arranque

¿En qué casos un disquete de arranque es una opción relevante?

Normalmente, el ordenador arranca desde el CD de Oralux.

Algunos ordenadors no pueden arrancar el CD ROM o configurar la BIOS es una operación difícil para el usuario.

El disquete de arranque es muy provechoso para proveer a Oralux los "cheat codes" que se requieran.

¿Cómo crear este disquete de arranque?

Con MS-Windows

knoblind.bat es un archivo de procesamiento por lotes de MS DOS que puede crear un disquete de arranque o actualizarlo.

Con Oralux

Introduce un disquete y teclea:

dd if=/cdrom/KNOPPIX/boot.img of=/dev/fd0 bs=18k

Después, actualiza el archivo /mnt/floppy/syslinux.cfg con el "cheatcode" deseado. La segunda línea del archivo es la concerniente: la línea APPEND.

Síntesis de voz

¿Qué sintetizadores hay en el CD?

El CD incluye un motor texto-a-voz (TTS) en inglés libre: Flite (de la universidad de Carnegie Mellon).

También incluye EFM de Pierre Lorenzon para los idiomas inglés y francés. está basado en Festival (de la Universidad de Edimburgo) y MBROLA (Thierry Dutoit, Universite de Mons).

Parlemax es un servidor de síntesis de voz en Francés basado en LLiaphon y MBROLA. Proviene del equipo Biglux.

Multispeech es un servidor de síntesis de voz basado en Ru_tts para el Ruso, y en freephone y MBROLA para el Inglés.

Multispeech puede cambiar automáticamente de una síntesis a otra. Igor B. Poretsky es el autor de Multispeech y Ru_tts.

¿Qué es el programa DECtalk?

Si compraste el programa DECtalk (4.61) , Oralux lo instalará de tu disco duro.

Oralux puede también instalar la versión 5.0.

DECtalk está disponible en varios idiomas y en especial en francés.

Para más información, aquí está la URL con el formulario de compra:

Obviamente, no tenemos ningún interés financiero en informar del programa DECtalk.

¿Y qué hay de la instalación del programa DECtalk?

Para que el CD pueda detectar el programa DECtalk, has de crear un directorio en una de tus particiones. Deberías evitar una partición NTFS: con ella, el número serie de DECtalk se pedirá en cada arranque. Porque a día de hoy, Oralux no puede escribir en particiones NTFS.

El nombre del directorio esperado es dtk.

Todo en minúsculas.

Este directorio debe estar ubicado en el nivel más alto (raíz) de la partición.

Por ejemplo, si /dev/hda1 está montada en /var, Oralux encontrará DECtalk en /var/dtk, pero no en /var/foo/dtk.

En el directorio dtk, coloca el paquete de DECtalk: el nombre esperado es algo así como linux.....tar.gz.

Si tienes otro paquete de DECtalk, colócalo ahí también. Oralux puede instalar el paquete opcional en Francés, o el paquete en español o en alemán.

¿Puede Oralux manejar un sintetizador externo?

Oralux incluye varios controladores para sintetizadores externos.

Estos controladores provienen de Emacspeak-ss.

Si tu tarjeta de sonido funciona bajo Oralux, puedes establecer tus preferencias usando el menú de audio.

Por ejemplo, tu seleccionarás tu sintetizador externo de una lista.

Mi tarjeta de sonido no funciona, ¿como inicio mi sintetizador externo?

El cheatcode tts

Por ejemplo, la siguiente línea ingresada al arranque, inicia Emacspeak con DoubleTalk LT:

knoppix tts=doubletalk

Por defecto se asume que el sintetizador estará conectado al primer puerto serial (ttyS0).

Para seleccionar el segundo puerto serial, deberías ingresar:

knoppix tts=doubletalk,ttyS1
Controladores disponibles

Aquí se encuentra una lista de los controladores incluidos.

Primero, se ingresa la palabra clave para el truco tts, luego el nombre del sintetizador.

  • accent: Accent SA
  • braillenspeak: Braille Lite, Braille 'n Speak, Type 'n Speak
  • ciber: Ciber 232, Ciber 232 Plus
  • dtk-exp: DECtalk Express
  • dtk-mv: DECtalk Multivoice, DECtalk 3
  • apollo: Dolphin Apollo
  • doubletalk: DoubleTalk LT, LiteTalk
  • doubletalkpc: DoubleTalk PC
  • pchablado: PC Hablado notebook

¿Cómo configurar la velocidad de la voz en emacspeak?

Para modificar lav velocidad del habla:

C-e d 1

1 para obtener una voz muy lenta,

C-e d 9

9 para obtener una voz muy rápida.

C-e significa: pulsa y mantén pulsada la tecla de CONTROL, y pulsa la tecla e. Suelta ambas teclas. Pulsa la tecla d, después 9.

Cualquier comando de emacspeak empieza con C-e. d está realcionada con el motor de voz, el primero de los cuales fue DECtalk.

Grabando preferencias y documentos.

¿Cómo grabo mis preferencias?

Probablemente querrás grabar el volumen, la configuración del teclado, y cualquier opción que hallas elegido en el menú inicial...

Lo mismo es aplicable a otros archivos, como la configuración de Internet.

saveconfig puede grabar estos archivos al disco duro, a un disquete, a una tarjeta de memoria USB...

En los arranques siguientes, los parámetros serán localizados automáticamente.

Aquí está la secuencia bajo Emacspeak.

Inicia un buffer the shell

M-x shell

Inicia el script saveconfig

saveconfig

Luego, indicarás qué archivos de configuración han de grabarse, y en dónde.

¿Cómo grabar mis documentos?

El directorio en el que puedes grabar tus documentos fácilmente es /home/knoppix.

Por defecto, este directorio está en un disco RAM. ¡En cuánto apagues el ordenador, el directorio se perderá!

De hecho, si quieres, este directorio /home/knoppix puede grabarse en tu disco duro, o en una tarjeta de memoria USB.

En el siguiente arranque, el directorio será localizado automáticamente.

Aquí está la secuencia bajo Emacspeak.

Inicia un buffer de shell

M-x shell

Inicia el script mkpersistenthome

mkpersistenthome

Elige dónde quieres que se grabe tu directorio.

Después, sal de emacspeak: C-x C-c

o sal de YASR.

Y reinicia el ordenador

Preguntas varias.

¿Cómo instalar Oralux en el disco duro?

La instalación real al disco duro es una función por desarrollar.

Aún así, es posible copiar el archivo comprimido del CD a cualquier partición fat o ext2 (pero no NTFS). Se requerirán menos de 700 megas. Oralux se comportará igual que corriendo desde el CD. No será posible añadir o modificar los programas: esto puede considerarse como una ventaja.

Hay dos etapas distintas:

  1. Crear un directorio KNOPPIX. Este directorio debe ser de nivel superior: ej. con MS-Windows, C:\KNOPPIX is correcto, C:\MyFolder\KNOPPIX no lo es.
  2. Copia el archivo KNOPPIX/KNOPPIX desde el CD de Oralux al nuevo directorio.
  3. Copy the file KNOPPIX/KNOPPIX from the Oralux CD to the new directory.

El CD de Oralux o el disquete de arranque pueden necesitarse para arrancar Oralux desde el disco duro.

¿Por qué un gallo como insignia?

Así es, la insignia de Oralux es un gallo cantando delante del sol.

El canto del gallo recuerda la palabra Oral y como el gallo canta al amanecer, recuerda a Lux que significa Luz.

Y no es un gallo anónimo: ¡Este es Galloralux!

Documentación

Convenciones

Unas pocas palabras, si no estás todavía familiarizado con la sintaxis C-x o M-x de Emacs.

C-x significa: mantén apretada la tecla Control y presiona la tecla x, después suelta ambas.

M es la también denominada META tecla que es la tecla ALT izquierda.

M es también la tecla Escape. La tecla Escape no hay que mantenerla apretada mientras se presiona la siguiente tecla.

Info

La documentación de Emacs y Emacspeak está principalmente en el formato Info. Este formato estructura la documentación, y permite al lector ir de nodo a nodo. Para aprender rápidamente los pocos comandos de Info, simplemente presiona C-h i
